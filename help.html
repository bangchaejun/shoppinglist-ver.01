Shopping List 도움말

쇼핑 리스트 앱의 각 기능을 쉽게 이해하고 사용할 수 있도록 도와드립니다.

예산 설정

예산을 설정하려면 "예산 설정 (₩)" 입력 필드에 금액을 입력하고, "설정" 버튼을 클릭하세요. 예산은 원화 단위로 입력할 수 있습니다.

아이템 추가

구매할 아이템을 추가하려면 "무엇을 구매할까요?" 입력 필드에 아이템 이름을 입력하고, "추가" 버튼을 클릭하세요. 같은 이름의 아이템은 중복으로 추가되지 않습니다.

아이템 구매

리스트에 있는 아이템을 구매하려면 각 아이템 옆에 있는 "구매" 버튼을 클릭하고, 아이템의 가격을 입력하세요. 입력한 금액이 총 지출 금액에 반영됩니다.

아이템 삭제

리스트에서 아이템을 삭제하려면 아이템 옆에 있는 체크박스를 선택한 후, "선택 항목 삭제" 버튼을 클릭하세요.

쇼핑 내역

구매한 아이템과 그 가격은 쇼핑 내역에 저장됩니다. 쇼핑 내역은 로컬 스토리지에 저장되므로 페이지를 새로고침해도 유지됩니다.

내역 삭제

쇼핑 내역에서 특정 아이템을 삭제하려면 내역 옆에 있는 체크박스를 선택한 후, "선택 항목 지우기" 버튼을 클릭하세요.

목록 공유

쇼핑 내역을 다른 사람과 공유하려면 "쇼핑목록공유" 버튼을 클릭하세요. 쇼핑 내역이 클립보드에 복사됩니다.

초기화

모든 값을 초기화하려면 "초기화" 버튼을 클릭하세요. 예산, 총 지출 금액, 쇼핑 리스트, 쇼핑 내역이 모두 초기화됩니다.
